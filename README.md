# 🍽️ Restaurant Bill Splitter (Python)

## 📌 Overview
This is a simple Python program that calculates a restaurant bill, applies a tip, and splits the total amount evenly among a group of friends.

The project demonstrates fundamental Python programming concepts such as:
- Variables and data types
- Arithmetic operations
- Incremental totals using a running total
- User-friendly formatted output
- Rounding for financial accuracy

---

## ⚙️ Features
- Calculates total cost of multiple items (appetizers, main course, desserts, drinks)
- Applies a configurable tip percentage
- Splits the bill evenly among a specified number of people
- Displays a clean, readable summary of the bill

---

## 🧮 Example Calculation
- Subtotal = Sum of all items  
- Tip = 25% of subtotal  
- Final Total = Subtotal + Tip  
- Cost per Person = Final Total ÷ Number of Friends
